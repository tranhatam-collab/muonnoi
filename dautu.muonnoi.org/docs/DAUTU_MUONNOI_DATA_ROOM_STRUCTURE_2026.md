# CẤU TRÚC DATA ROOM — DAUTU.MUONNOI.ORG

Phiên bản: v0.1 — 2026  
Trạng thái: Draft Proposed — Chờ Founder khóa  
**CHỈ mở cho nhà đầu tư đủ điều kiện và sau legal review.**

---

## 1. Định nghĩa

Data Room là không gian bảo mật chứa tài liệu nhạy cảm về dự án, chỉ dành cho nhà đầu tư đã qua thẩm định.

---

## 2. Điều kiện truy cập

| Điều kiện | Yêu cầu |
|---|---|
| Form quan tâm | Đã submit đầy đủ |
| Manual review | Đội phụ trách duyệt |
| Quốc gia | Không thuộc danh sách cấm/sanction |
| Tư cách nhà đầu tư | Accredited / Sophisticated / Qualified (tùy quốc gia) |
| KYC/KYB | Hoàn thành nếu yêu cầu |
| AML | Pass screening |
| NDA | Đã ký (nếu yêu cầu) |

---

## 4. Nội dung bảo mật

| Loại tài liệu | Cấp độ | Vị trí |
|---|---|---|
| Pitch deck | Data Room only | 01. Company Overview |
| Financial projections | Data Room only | 03. Financials |
| Cap table | Data Room only | 01. Company Overview |
| Legal memo | Data Room only | 02. Legal |
| Term sheet | Data Room only | 06. Investment Terms |
| Hợp đồng đầu tư | Data Room only | 06. Investment Terms |
| Tài khoản thanh toán chính thức | Data Room only | 08. Payment & Entity |
| Quy trình xác nhận giao dịch | Data Room only | 08. Payment & Entity |

**KHÔNG có trên public page:** Định giá, cổ phần, ROI, lợi nhuận dự kiến.

---

## 5. Bảo mật Data Room

- Truy cập qua invite link + login.
- Watermark tài liệu theo tên nhà đầu tư.
- Không cho phép download (viewer mode) hoặc giới hạn download.
- Audit log: ai xem gì, khi nào.
- Revoke access nếu nhà đầu tư vi phạm NDA.
- 2FA cho tài khoản Data Room.

---

## 6. Quy trình mở Data Room

```
Form Interest
    → Manual Review (5 ngày)
        → KYC/KYB/AML (nếu cần)
            → NDA (nếu cần)
                → Invite Data Room
                    → Pitch / DD / Deal
```
