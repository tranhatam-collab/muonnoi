## 1. Nguyên tắc định giá

```
Muon Noi Ecosystem Value =
```

## 2. Ba lớp định giá

### Lớp 1 — Asset Value

Dùng cho giai đoạn **chưa có doanh thu**.

| Thành phần | Mô tả |
|---|---|
| Domain/subdomain | Danh mục subdomain thương hiệu |
| Nội dung SEO đa ngôn ngữ | Content đa ngôn ngữ đã index |
| Codebase | Source code, architecture, system design |
| UI system | Design system, component library |
| Tài liệu pháp lý/sản phẩm | Docs, specs, legal framework |
| Thương hiệu | Brand recognition, positioning |
| Cộng đồng ban đầu | Early adopters, community members |
| Pháp nhân | Corporate entities, IP ownership |
| Tài sản số, workflow, docs | Internal processes, automation |

### Lớp 2 — Traction Value

Dùng cho **seed/pre-seed**.

| Metric | Ý nghĩa |
|---|---|
| User đăng ký | Số lượng sign-ups |
| Verified users | Số lượng đã xác minh danh tính |
| Retention | Tỷ lệ giữ chân người dùng |
| Quest completions | Số quest đã hoàn thành |
| Gift transactions | Số giao dịch quà tặng |
| Host/mentor supply | Số host/mentor đã đăng ký |
| Paid verification | Số người dùng trả phí xác minh |
| Paid plans | Số subscription |
| Organic SEO traffic | Traffic tự nhiên từ SEO |

### Lớp 3 — Revenue Multiple

Khi có doanh thu: `ARR × multiple`

| Yếu tố ảnh hưởng multiple | Mô tả |
|---|---|
| Tốc độ tăng trưởng | Growth rate MoM/YoY |
| Gross margin | Biên lợi nhuận gộp |
| Compliance risk | Mức độ rủi ro pháp lý |
| Retention | Churn rate, LTV |
| Payment quality | Chargeback rate, fraud |
| Defensibility | Khả năng bảo vệ thị phần |
| International expansion | Tiềm năng mở rộng quốc tế |

## 3. Khung định giá theo giai đoạn

### Giai đoạn 0 — Founder Build / Pre-revenue

**Trạng thái:** Plan + prototype + docs + subdomain + legal structure

**Định giá nội bộ tham khảo:** 500.000 – 2.000.000 USD

**Ghi chú:** Chỉ dùng nội bộ, không public.

### Giai đoạn 1 — MVP có user thật

**Điều kiện:**
- 5.000 – 20.000 users
- 1.000+ verified users
- payment/verification chạy thật
- no critical legal gap
- retention đo được

**Định giá tham khảo:** 2.000.000 – 8.000.000 USD

### Giai đoạn 2 — Pilot revenue

**Điều kiện:**
- Có doanh thu xác minh (quà, subscription, event, partner)
- Có 3–5 sản phẩm live
- Báo cáo tài chính rõ

**Định giá tham khảo:** 8.000.000 – 25.000.000 USD

### Giai đoạn 3 — Scale đa quốc gia

**Điều kiện:**
- Nhiều thị trường
- Compliance theo quốc gia
- Payment localization
- User growth thật
- Doanh thu định kỳ

**Định giá tham khảo:** 25.000.000 – 100.000.000+ USD
