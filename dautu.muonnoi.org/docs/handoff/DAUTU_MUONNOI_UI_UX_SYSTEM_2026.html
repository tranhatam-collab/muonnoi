🎨 UI/UX SYSTEM

Quy tắc thiết kế UI/UX bắt buộc cho dautu.muonnoi.org — đảm bảo consistency, trust, và doctrine-first experience

1. DESIGN LANGUAGE

1.1. Design principles

  • Premium nhưng không hào nhoáng — Professional, trustworthy
  • Calm — Not chaotic, not overwhelming
  • Sáng — Clear, transparent, honest
  • Sâu — Substance over style
  • Rõ — Unambiguous, direct
  • Tin cậy — Reliable, consistent

1.2. Anti-patterns

Không được:

  • Fintech hype design
  • Dashboard crypto style
  • Startup pitch-deck UI
  • Get-rich quick visual language
  • Urgency/scarcity tactics
  • Over-animated interfaces

7. MOTION SYSTEM

7.1. Motion principles

  • Light motion only — Subtle, not distracting
  • Purposeful — Motion must serve a purpose
  • Fast — 200-300ms duration
  • Smooth — Ease-in-out timing

7.2. Allowed animations

  • Hover effects (color change, slight scale)
  • Scroll reveal (fade in, slide up)
  • Section transitions (fade in/out)
  • Button interactions (press effect)

7.3. Forbidden animations

Không được:

  • Bouncing elements
  • Flashing/pulsing effects
  • Parallax (causes motion sickness)
  • Auto-playing carousels
  • Countdown timers (urgency tactics)

7.4. Reduced motion

  • Respect prefers-reduced-motion
  • Disable all animations when reduced motion is on
  • Provide static alternatives

8. ICON SYSTEM

8.1. Icon style

  • Line icons — Clean, modern
  • Consistent stroke width — 2px
  • Minimal detail — Simple, recognizable
  • Professional — Not cartoonish

8.2. Icon usage

  • Use icons for visual hierarchy
  • Don't overuse icons
  • Consistent sizing (16px, 24px, 32px)
  • Proper spacing around icons

9. IMAGERY SYSTEM

9.1. Image style

  • Professional photography — Real people, real settings
  • Authentic — Not stock-looking
  • Calm composition — Not chaotic
  • Consistent color grading — Muted, professional tones

9.2. Image optimization

  • WebP format with JPEG fallback
  • Responsive images (srcset)
  • Lazy loading below-fold
  • Compression (quality 80-85)
  • Proper alt text

10. ACCESSIBILITY

10.1. Color contrast

  • Text contrast: 4.5:1 minimum
  • Large text contrast: 3:1 minimum
  • Interactive elements: 3:1 minimum
  • Focus indicators: Clear visible focus states

10.2. Touch targets

  • Minimum size: 44x44px
  • Spacing: 8px between targets
  • Padding: Adequate padding for touch

10.3. Keyboard navigation

  • All interactive elements keyboard accessible
  • Logical tab order
  • Visible focus indicators
  • Skip to main content link

10.4. Screen readers

  • ARIA labels for interactive elements
  • Alt text for all images
  • Semantic HTML structure
  • Form labels properly associated

11. MOBILE UX

11.1. Mobile navigation

  • Hamburger menu — Collapsible navigation
  • Sticky CTA — Primary CTA always visible
  • Back navigation — Clear back buttons
  • Touch-friendly — Large touch targets

11.2. Mobile content

  • Progressive disclosure — Show key info first
  • Collapsible sections — Expandable content
  • Sticky headers — Section headers visible
  • Optimized reading — Readable font sizes
