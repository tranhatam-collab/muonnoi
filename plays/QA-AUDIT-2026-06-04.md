## 3. Lỗi SDK / Runtime (game crash hoặc không lưu điểm)

Các game dùng `../../plays-sdk.js` (root) gọi sai API so với `PlaysSDK` đã export:

| API gọi | API thực tế | Game bị ảnh hưởng |
|---------|-------------|-------------------|
| `PlaysSDK.saveScore(…)` | Không tồn tại (tên đúng: `saveGuestScore`) | game-064, simon-says, duoi-bat, tranh-bom, pixel-art, hit-tho, nho-vi-tri… |
| `PlaysSDK.showGameOver(id, score, {level}, cb)` | Signature object `{gameId, score, level, best, isNewBest, token, onRestart, onBack}` | Tất cả game dùng root SDK trừ nho-day, tinh-nham |
| `PlaysSDK.showTutorial(…)` | Không tồn tại (tên đúng: `buildTutorial`) | Tất cả game dùng root SDK |

## 4. Lỗi Giao diện / UI

- **Hai bộ CSS song song**: `assets/plays.css` (biến `--card`, `--line`) và `styles/plays.css` (biến `--surface`, `--border`). Giao diện stub và game thực không nhất quán.
- **catalog.js** ở root không được `index.html` chính sử dụng; thay vào đó dùng `assets/catalog-lite.js`. Dữ liệu 100 game trong `catalog.js` có thể đã lỗi thời.

## 5. Hành động đã thực hiện

1. **Fix SDK runtime** (`plays-sdk.js` root + `assets/plays-sdk.js`):
   - Thêm backward-compat aliases: `saveScore`, `showTutorial`, `showGameOver` overload.
   - Thêm `MNPlays.showGameOver` vào `assets/plays-sdk.js`.
2. **Fix gameplay** (thêm điều kiện thua / endGame):
   - `game-064`: hết moves → endGame.
   - `pixel-art`: sai 3 lần → endGame.
   - `cau-tu-doi`: sai 3 lần → endGame.
   - `mat-ma-chu`: sai 3 lần → endGame.
   - `game-098` (Tháp Hà Nội): hết maxMoves → endGame.
   - `sudoku-mini`: điền sai 3 lần → endGame.
3. **Xử lý 7.390 stub**: Đã chuyển `status` từ `live` → `wave1` (planned) trong `assets/catalog-lite.js`. Chỉ còn **106 game live** trên catalog, tương ứng với các game có gameplay thực.

## 6. Cần làm tiếp

- **Các game board** (`co-caro`, `co-vua-mini`, `game-100`) vẫn cho phép thắng AI → qua level. Cần quyết định: giữ nguyên (vẫn có thể thua AI) hay đổi thành endless mode hoàn toàn.
- **44 game live còn lại** trong catalog cần kiểm tra xem có phải stub không khớp tên thư mục.
