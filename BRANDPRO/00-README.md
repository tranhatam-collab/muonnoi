# Brandpro — Bộ Kit Chuẩn Hóa Thương Hiệu Toàn Cầu & Tech Product v2.0

> Phiên bản: 2.0 · Ngôn ngữ chính: Tiếng Việt · Áp dụng cho: thương hiệu mới, tái định vị, tech startup, SaaS, product studio, holding brand.

---

## 1. Brandpro v2.0 là gì

Brandpro là **bộ tài liệu khung 20 file** giúp một team đi từ:

```
Tên thương hiệu chưa kiểm chứng
    -> Bộ nhận diện hoàn chỉnh
    -> Đăng ký pháp lý
    -> Vận hành đa kênh, đa sản phẩm
    -> Design system + developer handoff
    -> Đo lường được
```

Mỗi file là **template có thể điền trực tiếp**, kèm rubric chấm điểm, checklist, ví dụ minh họa. Đầu ra mỗi file là một artefact có thể bàn giao.

**Khác biệt v2.0 so với v1.1:**
- Thêm 4 module cho **tech product**: Product Brand System, Design System & Dev Handoff, Brand Architecture, Competitive Moat.
- Mở rộng Verbal Identity + Visual Identity + Trackers cho **product layer** (UI copy, design tokens, code naming).
- Timeline mở rộng từ 90 -> **120 ngày** cho dự án có design system và product branding.

---

## 2. Triết lý nền (7 nguyên tắc)

1. **Logic trước Mỹ thuật** — quyết định chiến lược (tên, định vị, va chạm) trước khi vẽ logo.
2. **Một sự thật duy nhất** — tên brand chỉ có một cách viết, mọi điểm chạm phải nhất quán (cả trong code).
3. **Dạy lại cho cả người và máy** — entity SEO không kém phần quan trọng so với visual identity.
4. **Song ngữ là transcreation, không phải translation** — mỗi ngôn ngữ một bộ messaging riêng.
5. **Brand book chỉ có giá trị nếu được enforce** — governance là phần dài hạn nhất.
6. **Product là điểm chạm brand quan trọng nhất** — UI copy, error message, onboarding đều là brand.
7. **Design system = brand system** — không tách rời visual identity và code implementation.

  \- File 06 -> 07 -> 08 -> 17 (design tokens)
  \- Cong duyet: Brand Book v1 + Design Token v1 duoc duyet

Pha 4 — PRODUCT, LOCALIZATION & DIGITAL (Day 57 -> Day 90)
  \- File 09 -> 10 -> 16 -> 17 (dev handoff)
  \- Cong duyet: Website song ngu + entity SEO live + Product Copy lock + Design System v1

Pha 5 — PHONG THU & VAN HANH (Day 91 -> Day 120)
  \- File 11 -> 12
  \- Cong duyet: Don TM nop + Brand Guardian duoc bo nhiem

Sau Day 120, vao nhip **Quarterly Audit** (file 12) — vinh vien.

**Scale theo nhu cau:**
- **Lean startup** (< 200tr, 1 founder): dung 8 file (01, 02, 03, 04, 05, 06, 07, 12) + 8 artefact. Bo qua 16-19.
- **Standard SME / Single SaaS** (200-800tr): dung 14 file (them 08, 09, 10, 11, 13, 14). Bo qua 18-19.
- **Premium / Holding / Multi-product** (> 800tr): full 20 file.

## 6. Cach dien template

- Moi file co 3 vung: **HUONG DAN** (doc mot lan) / **TEMPLATE** (dien vao) / **VI DU** (tham chieu khi bi).
- Quy uoc ky hieu:
  - `[ ]` checkbox chua lam · `[x]` da lam · `[~]` dang lam · `[!]` blocker
  - `<<...>>` placeholder phai thay bang noi dung that
  - `TODO` ghi chu cong viec, `FIXME` ghi loi can sua, `DECISION` can quyet dinh truoc khi tiep
- Moi quyet dinh lon phai co dong **Nguoi duyet:** va **Ngay duyet:** o cuoi section.
- Khong xoa noi dung HUONG DAN sau khi dien — de team moi onboarding hieu logic.
