## Bilingual Strategy

### Strategy

**Strategy:** 1-to-1 mapping between VI and EN terms.

**Rationale:**
- Consistent translation across all surfaces
- No self-translation
- Context-consistent translation
- Quality translation

**Process:**
1. Create MASTER LANGUAGE CODEX
2. Map all VI terms to EN terms 1-to-1
3. Use MASTER LANGUAGE CODEX for all translations
4. No self-translation allowed
5. Context check for all translations

## Bilingual Guidelines

### Guidelines

**VI (Vietnamese):**
- Use natural Vietnamese
- Use full diacritics
- Use correct spelling
- Use appropriate tone (polite, friendly)
- Avoid slang (unless appropriate)
- Use terms from MASTER LANGUAGE CODEX

**EN (English):**
- Use natural English
- Use correct grammar
- Use appropriate tone (friendly, professional)
- Avoid slang (unless appropriate)
- Use terms from MASTER LANGUAGE CODEX
- Be consistent with Vietnamese brand voice
