## 1. muonnoi.org (root) — CRITICAL

**Files:** `index.html`, `app.html`, `admin.html`

### Issues
- **No `meta description`** — Google will auto-generate snippets from body text
- **No Open Graph tags** — No social sharing cards (Facebook, LinkedIn, Zalo, etc.)
- **No Twitter Card tags** — No rich Twitter sharing
- **No `link rel="canonical"`** — Duplicate content risk across subdomains
- **No `script type="application/ld+json"`** — No structured data for rich results
- **No `meta name="robots"`** — Implicit `index,follow`, but explicit is safer
- **No favicon** — Browser tab and search results lack branding
- **No `manifest.json`** — No PWA support
- **No `theme-color`** — Mobile browser chrome unbranded
- **No `hreflang`** — Bilingual content (VI/EN) not signaled to Google

### Impact
The root domain is the canonical brand entry point. It is currently invisible to search engines for any targeted queries.

### Fix Priority: P0

## Cross-Cutting Issues

### A. OG Images Using SVG
**Affected:** `ai.muonnoi.org`, `cuocsong.muonnoi.org`, `docs.muonnoi.org`  
**Fix:** Convert all OG images to PNG 1200×630 px. Facebook, LinkedIn, Zalo, Twitter do not render SVG in link previews.

### B. Missing Structured Data (JSON-LD)
**Affected:** `muonnoi.org` (root), `www.muonnoi.org`, `app.muonnoi.org`, `cuocsong.muonnoi.org`  
**Fix:** Add `WebSite` + `Organization` schema to all root index pages.

### C. No robots.txt at Root
**Affected:** `muonnoi.org` root  
**Fix:** Create `/robots.txt` with:
```
User-agent: *
Allow: /
Sitemap: https://www.muonnoi.org/sitemap.xml
```

### D. Admin Pages Not Noindexed
**Affected:** `muonnoi.org/admin.html`, `nguoiviet.muonnoi.org/admin/`  
**Fix:** Add `<meta name="robots" content="noindex,nofollow">` to all admin/login dashboards.

### E. No XML Sitemap at Root
**Affected:** `muonnoi.org` root  
**Fix:** Create `sitemap.xml` referencing `index.html`, `app.html`, and linking to subdomain sitemaps via sitemap index.

### F. H1 Tag Usage
**Affected:** `muonnoi.org/index.html` — `<h1>` is inside `.brand` div but page also has `<h2>` cards. The `<h1>` text "Muôn Nói · Muon Noi" is good.  
**Note:** `app.html` and `admin.html` both use `<h1>` for branding — acceptable since they are app interfaces, not content pages.
