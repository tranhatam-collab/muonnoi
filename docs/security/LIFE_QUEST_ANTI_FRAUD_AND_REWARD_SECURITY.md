# LIFE QUEST ANTI FRAUD AND REWARD SECURITY · MUONNOI 2026

## Principle
Rewards follow real value only.

Muonnoi must not create fake play-to-earn incentives, speculative tokens, or rewards that look like investment returns.

## Proof anti-fraud controls
- AI image/video pre-review
- GPS consistency check with consent
- timestamp consistency check
- host signature
- QR check-in where applicable
- peer review
- random audit
- appeal process
- trust penalty for abuse

## Reward rules
- XP is not money.
- Trust cannot be bought.
- Credits may only be exchanged under explicit product rules.
- Earning comes from real work, real hosting, real mentorship, real tasks or real contribution.
- No guarantee of income.
- No ROI language.

## Abuse scenarios
- fake location
- recycled photo
- coordinated false validation
- host self-dealing
- duplicate proof
- reward farming
- identity farming

## Required before pilot scale
- proof status model
- appeal flow
- audit log
- abuse reporting path
- host verification policy
- reward disclosure language
