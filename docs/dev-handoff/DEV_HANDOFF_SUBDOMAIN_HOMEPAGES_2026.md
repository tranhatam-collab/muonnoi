# DEV HANDOFF SUBDOMAIN HOMEPAGES · 2026

Before building a subdomain homepage:
1. Confirm Cloudflare project/domain.
2. Confirm source tree.
3. Confirm whether a near-complete version already exists.
4. Apply shared structure from `MUONNOI_SUBDOMAIN_HOMEPAGE_PACKAGE_2026.md`.
5. Run live route check before public CTA.

Do not create new projects when an existing mapped project already owns the domain.
