# Muonnoi Store Submission Requirements

Date: 2026-05-11
Scope: Apple App Store + Google Play readiness.

## Required before submission

1. App name, subtitle and description approved.
2. Privacy policy URL live.
3. Support URL live.
4. Account deletion path documented.
5. UGC moderation policy documented.
6. Abuse report flow functional.
7. Permission copy reviewed.
8. Screenshots prepared for required device sizes.
9. Age rating answered consistently.
10. Test account available for review.

## App Store review risks

- user-generated content without moderation
- unclear location permission
- camera/photo use without clear reason
- family/child safety claims without policy
- reward language that implies gambling, token or financial return

## Required copy posture

Allowed:
- "nhiệm vụ đời sống"
- "bằng chứng thật"
- "uy tín"
- "giá trị thật"

Not allowed:
- "chơi là kiếm tiền"
- "đảm bảo thu nhập"
- "token"
- "đầu tư"
- "lợi nhuận"

## Store launch gate

No store submission until:
- release runbook exists
- incident response exists
- privacy matrix approved
- mobile release gates pass
- Product signs public description
- Platform signs build identity and rollback readiness
