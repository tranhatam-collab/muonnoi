# CUOC SONG MUONNOI PUBLIC SITE CONTENT MAP · 2026-05-13

## Purpose

This file turns the Drive source package into a claim-safe site map for `cuocsong.muonnoi.org`.

It is a content blueprint, not a deployment record. No route is release-ready until the QA checklist passes.

## Global Rules

- Vietnamese pages use full Vietnamese with diacritics.
- English pages are complete pages, not short summaries.
- Language labels must be written as `Tiếng Việt` and `English`.
- Every page links back to `https://www.muonnoi.org/`.
- External subdomain links are allowed only when the DNS matrix marks the target host as link-safe.
- If a target host is not link-safe, use a root internal route or explain the module as planned.
- No page contains payment, donation, price, investment, visa, legal, medical, therapy or immigration service claims.
