# PROOF REVIEW AND REWARD RULES

Proof review must use:
- clear task criteria
- optional location only with consent
- host/community validation where applicable
- audit log
- appeal path

Reward rules:
- XP is not money.
- Trust cannot be bought.
- Credits require explicit rules.
- No guaranteed earning.
- No investment-return language.
- Real rewards must follow real work, real hosting, real mentoring or real contribution.
