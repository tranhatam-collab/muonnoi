# QUEST CONTENT TEMPLATE v1
> **Purpose:** Standard format for all Muonnoi quest content
> **Scope:** Applies to Đường Muôn Nơi, Học Đời, Family Mission, and all future quest systems
> **Status:** LOCKED — DO NOT ALTER WITHOUT FOUNDER APPROVAL
> **Locked By:** Founder, 2026-06-05

---

## MANDATORY FIELDS

Every quest file MUST include all fields below. Empty fields are not allowed. Use "N/A" if genuinely not applicable.

---

### 1. METADATA

```yaml
quest_id:        "unique-uuid-v4"          # UUID v4, generated by CMS
system:          "dulich|hoctap|family|..." # Which quest system
slug:            "keo-chai-dalat"           # URL-friendly, viết thường, dấu gạch ngang
city:            "Đà Lạt"                   # City name in Vietnamese
region:          "Lâm Đồng"                 # Province
language:        "vi|en|both"               # Content language
created_at:      "2026-06-05T00:00:00Z"     # ISO 8601 UTC
updated_at:      "2026-06-05T00:00:00Z"     # ISO 8601 UTC
version:         "1.0"                      # Template version
author_id:       "host-uuid-or-founder"     # Who wrote this quest
reviewer_id:     "reviewer-uuid"            # Who reviewed and approved
status:          "draft|review|active|paused|archived"
```

---

### 2. QUEST IDENTITY

| Field | Required | Description |
|-------|----------|-------------|
| **title_vi** | ✅ | Tên quest tiếng Việt, max 60 chars, không dấu chấm cuối |
| **title_en** | ✅ | Tên quest tiếng Anh, max 60 chars |
| **subtitle_vi** | ✅ | Mô tả 1 dòng, max 120 chars, giải thích "đây là gì" |
| **subtitle_en** | ✅ | English subtitle, max 120 chars |
| **intent** | ✅ | 1-2 câu: tại sao quest này tồn tại? Giá trị cho user là gì? |
| **who_fits** | ✅ | 3-5 đặc điểm người phù hợp ("bạn thích...", "bạn muốn...") |
| **who_not_fit** | ✅ | 2-3 đặc điểm người KHÔNG nên tham gia |

---

### 3. LOGISTICS

| Field | Required | Format |
|-------|----------|--------|
| **duration_min** | ✅ | Số phút (ví dụ: 180 = 3 giờ) |
| **duration_display** | ✅ | Text hiển thị ("3 giờ", "Nửa ngày") |
| **difficulty** | ✅ | `easy` / `medium` / `hard` |
| **min_age** | ✅ | Số tuổi tối thiểu (13 nếu không giới hạn) |
| **max_people** | ✅ | Số người tối đa (1, 2, 4, 6, 8, 12) |
| **group_type** | ✅ | `solo` / `pair` / `small_group` / `family` |
| **price_vnd** | ✅ | 0 nếu miễn phí. Nếu có phí: số VNĐ/người |
| **price_note** | ✅ | "Bao gồm..." hoặc "Không bao gồm..." hoặc "Miễn phí" |
| **best_time** | ✅ | Tháng/mùa tốt nhất ("Tháng 3-5", "Quanh năm") |
| **weather_dependency** | ✅ | `yes` / `no` + ghi chú nếu yes |
| **accessibility** | ✅ | `wheelchair_friendly` / `not_accessible` / `limited` + mô tả |

---

### 4. PROOF REQUIREMENTS

| Field | Required | Description |
|-------|----------|-------------|
| **proof_types** | ✅ | Array: `["photo", "text", "gps"]` — v0.1 chỉ hỗ trợ 3 loại này |
| **proof_description** | ✅ | User cần làm gì để submit proof? ("Chụp ảnh bạn đang...", "Viết 3 câu về...") |
| **proof_example** | ✅ | Ví dụ proof hợp lệ (text mẫu hoặc mô tả ảnh mẫu) |
| **proof_invalid_example** | ✅ | Ví dụ proof KHÔNG hợp lệ (ảnh chụp màn hình, text copy-paste...) |
| **gps_required** | ✅ | `true` / `false` |
| **gps_radius_m** | ✅ nếu gps_required=true | Bán kính chấp nhận GPS (ví dụ: 100m) |
| **gps_target_lat** | ✅ nếu gps_required=true | Latitude mục tiêu |
| **gps_target_lng** | ✅ nếu gps_required=true | Longitude mục tiêu |
| **gps_target_name** | ✅ nếu gps_required=true | Tên địa điểm GPS ("Đỉnh Langbiang") |

---

### 5. HOST ROLE

| Field | Required | Description |
|-------|----------|-------------|
| **host_role_summary** | ✅ | Host làm gì trong quest này? ("Dẫn đường", "Giới thiệu văn hóa", "Chuẩn bị bữa ăn") |
| **host_prep_required** | ✅ | `true` / `false` + mô tả nếu true |
| **host_language** | ✅ | Ngôn ngữ host giao tiếp (`vi`, `en`, `vi+en`) |
| **host_meet_location** | ✅ | Điểm hẹn (địa chỉ cụ thể + GPS) |
| **host_meet_time** | ✅ | Giờ hẹn mặc định ("08:00 sáng") |

---

### 6. SAFETY NOTES

| Field | Required | Description |
|-------|----------|-------------|
| **safety_level** | ✅ | `low` / `medium` / `high` |
| **safety_notes** | ✅ | Cảnh báo cụ thể ("Đường dốc, cần giày đế cao", "Nắng gắt, mang theo nước") |
| **required_gear** | ✅ | Array: `["giày đi bộ", "nón", "nước uống"]` — có thể trống nhưng không bỏ trường |
| **emergency_contact** | ✅ | Số hotline hoặc "Gọi host qua app" |
| **weather_cancel_policy** | ✅ | "Hủy nếu mưa to", "Vẫn đi nếu mưa nhỏ", v.v. |

---

### 7. NEARBY SUGGESTIONS

| Field | Required | Description |
|-------|----------|-------------|
| **before_quest** | ✅ | Gợi ý làm trước quest ("Ăn sáng ở...", "Tham quan... trước") |
| **after_quest** | ✅ | Gợi ý làm sau quest ("Uống cà phê ở...", "Mua đặc sản tại...") |
| **combo_with** | ✅ | Array quest_id có thể kết hợp (ví dụ: `["quest-uuid-2"]` ) |

---

### 8. WHAT USER RECEIVES

| Field | Required | Description |
|-------|----------|-------------|
| **xp_reward** | ✅ | Số XP (ví dụ: 50, 100, 200) |
| **trust_reward** | ✅ | Số Trust point (ví dụ: 10, 25) |
| **physical_reward** | ✅ | "Chứng nhận tay viết", "Món quà nhỏ từ host", "N/A" |
| **skill_tag** | ✅ | Array kỹ năng: `["thể lực", "văn hóa", "ẩm thực"]` |
| **reflection_prompt** | ✅ | 1 câu hỏi để user suy ngẫm sau quest ("Bạn cảm nhận gì về...?") |
| **next_quest_suggestion** | ✅ | Gợi ý quest tiếp theo (title + quest_id) |

## VALIDATION RULES

Before a quest can be marked `active`, it must pass:

1. **All mandatory fields present** — no empty required fields
2. **GPS coordinates valid** — lat ∈ [-90, 90], lng ∈ [-180, 180]
3. **Title length** — vi ≤ 60 chars, en ≤ 60 chars
4. **Price consistency** — if `price_vnd` > 0, `price_note` must explain what's included
5. **Proof types valid** — only from allowed set: `["photo", "text", "gps"]` for v0.1
6. **Safety notes non-empty** — at least 1 concrete warning
7. **Reflection prompt present** — must be 1 open-ended question
8. **Author and reviewer set** — cannot be anonymous
9. **Status transition valid** — `draft` → `review` → `active` only (no skip)

## BILINGUAL RULE

- `title_vi` + `subtitle_vi` + `intent` + `who_fits` + `who_not_fit` + `safety_notes` + `reflection_prompt` — **VIETNAMESE**
- `title_en` + `subtitle_en` — **ENGLISH**
- If `language: "both"` — all user-facing text must have both languages
- If `language: "vi"` — English fields can be empty but Vietnamese must be complete
- If `language: "en"` — reverse
