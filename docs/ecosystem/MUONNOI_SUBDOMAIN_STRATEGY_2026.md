# MUONNOI SUBDOMAIN STRATEGY · 2026

## Rule
Subdomains are product surfaces, not isolated brands.

Before publishing a subdomain:
- check Cloudflare project/domain
- confirm source folder
- confirm route owner
- use shared public language
- pass `docs/brand/MUONNOI_BRAND_ARCHITECTURE_AND_SUBDOMAIN_RULES_2026.md`
- pass `docs/brand/MUONNOI_BRAND_GUARDIAN_AND_QA_GATE_2026.md`
- link back to `muonnoi.org`, `docs.muonnoi.org`, `/security/`, `/guide/`

Naming rule:
- Each subdomain remains a Muonnoi module, not a separate brand.
- Public label pattern: `Muôn Nơi · <Module>` in Vietnamese and `Muon Noi · <Module>` in English.
