# MUONNOI VERBAL IDENTITY AND BILINGUAL MESSAGING · 2026

## Voice
Muonnoi speaks like:
- a serious community builder
- a careful product operator
- a privacy-aware technologist
- a human editor who respects real life

Voice axes:
- serious over playful
- warm over distant
- evidence-led over emotional hype
- humble but decisive

## Tone by context
Homepage:
- clear, warm, public, human

Docs:
- exact, direct, versioned, evidence-led

Security:
- calm, strict, factual

Investment:
- careful, non-promissory, no ROI language

Life Quest:
- grounded, purposeful, life-first

Mobile:
- concise, action-oriented, privacy-aware

Error or blocked state:
- name the problem
- name what is safe now
- name the next step

## Bilingual rules
Vietnamese pages:
- Vietnamese with diacritics is required.
- Do not remove accents from public text.
- Do not translate technical tokens blindly.

English pages:
- natural English, not literal Vietnamese translation.
- keep `Muon Noi` as ASCII brand form.
- explain `Muôn Nơi` only where needed.

Technical tokens:
- Keep original tokens when standard:
`API`, `CSP`, `CORS`, `D1`, `JWT`, `OG`, `JSON-LD`, `hreflang`, `x-default`.

## Brand copy QA
A page fails brand QA if:
- it says ready without evidence
- it has missing title or description
- it uses `VI/EN` labels
- it links to an unverified subdomain as primary CTA
- it uses investment promise language
- it uses decorative symbols instead of clear text
- it has more than one H1
