## 1. Brand core

### Master brand
- Vietnamese: **Muôn Nơi**
- English: **Muon Noi**
- Domain: `muonnoi.org`
- Compact mark: `MN` (icon only, never replacement for name)

### Meaning
Muôn Nơi carries two meanings at once:
- **many places**
- **muốn nói** as a spoken echo of wanting to speak authentically

### Core philosophy
**Nói từ muôn nơi. Nghe thấy khắp nơi.**

### Master tagline
- VI: **Hạ tầng số cho đời sống thật**
- EN: **Real life first. Real people first. Real value first.**

---

## 2. Brand positioning

Muôn Nơi is not a generic social network.  
It is not an e-commerce shell.  
It is not a crypto-like ecosystem.  
It is a **real-life-first digital infrastructure brand**.

The system must communicate:
- trust
- clarity
- calm confidence
- real life utility
- privacy-aware digital architecture
- cultural depth without nostalgia performance

---

## 3. Architecture hierarchy

### Master brand
- `muonnoi.org`
- `app.muonnoi.org`
- `docs.muonnoi.org`

### Sub-brand
- `nguoiviet.muonnoi.org`

### Feature subdomains
- work
- travel
- health
- creation
- learning
- family
- investment
- project
- community
- trust
- housing/community

### Naming lock
Use:
**Muôn Nơi · [Module Name]**

Do not:
- invent alternate separators
- create competing sub-brands casually
- use ASCII-only visible naming in public UI when Vietnamese is the main display language

## 4. Visual identity lock

### Core colors
- Azure `#3B7EFF`
- Whisper `#7FE0E5`
- Gold `#D4AF37`
- Charcoal `#0A0F14`

### Semantic colors
- Success `#10B981`
- Warning `#F59E0B`
- Error `#EF4444`

### Backgrounds
- White / light gray / border gray in light mode
- Charcoal / dark gray / border gray in dark mode

### Brand mood
The palette must feel:
- digital
- trustworthy
- premium
- calm
- modern

Not:
- playful
- neon
- crypto-flashy
- childish
- loud

---

## 5. Typography lock

### Display / headline
Inter Bold, with Roboto Bold as fallback

### Body
Inter Regular, with Roboto Regular as fallback

### Mono
JetBrains Mono

### Rules
- Vietnamese diacritics always preserved
- no visible text without accents when Vietnamese is primary
- bilingual content has equal dignity
- no hierarchy cheating by shrinking English too much
- code stays in English where necessary

---

## 6. Logo system lock

### Master mark
Two concentric circles with interlocking MN construction

### Lockups included
- horizontal logo light
- horizontal logo dark
- horizontal logo mono
- icon / mark light
- icon / mark dark
- favicon

### Usage
- `MN` mark is allowed in icon contexts only
- full brand name remains visible in normal public brand contexts
- do not replace “Muôn Nơi” with “MN” in body copy

### Sub-brand mark
Người Việt Muôn Nơi uses a related but warmer sub-brand mark with Vermillion, Teal, and Gold accents.

---

## 7. Component behavior lock

### Buttons
- Primary = Azure
- Secondary = Whisper
- Minimum touch target = 48px desktop / 44px mobile acceptable minimum
- always visible focus state

### Links
- underlined
- Azure by default
- never rely on color only

### Cards
- soft radius
- white / dark neutral backgrounds
- restrained shadow
- clear content hierarchy
- no unnecessary decoration

### Navigation
- sticky
- bilingual
- text-based language toggle
- never “VI/EN” shorthand in public UI

---

## 8. Người Việt Muôn Nơi sub-brand lock

This is not a separate company brand.  
It is a **structured sub-brand** inside Muôn Nơi.

### Role
Vietnamese diaspora identity, stories, journeys, mentorship, resources, and hubs

### Core tagline
- VI: **Đi xa để quay trở về**
- EN: **Journey far to return home richer**

### 7-stage model
1. Con người
2. Mắc kẹt
3. Môi trường
4. Va chạm
5. Trí tuệ
6. Hệ quả
7. Quay trở về

### Accent colors
- Vermillion
- Teal
- Gold
- supporting warm neutrals

No team may:
